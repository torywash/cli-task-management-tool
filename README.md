# CLI Task Management Tool
<hr>
This is a simple CLI (Command Line Interface) Task Management script, where you can mark events and important dates, with full descriptions, priorities, and editing.

## Purpose
<hr>
my goals for this are as follows:
* continue work on my planning and coding skills
* work on scripting, making something useful for myself

## Final Thoughts
<hr>
This was interesting. I thought about Professor Berns' thoughts of AI usage, and thought about it as a crutch instead of a resource. If I couldn't see a way around a problem, I had claude.ai point me in a direction to start working, copilot suggested changes and helped build on what I built. I didn't prompt copilot with anything, but only used the suggestions for what I was going to build. It sped up the entire process. I think this is a good step in my career, differentiating the usage of AI from a builder to a tool.

# How to Use
<hr>
1) Download the python file to somewhere on your computer
2) Using *CMD* or an *IDE* (such as VSCode), run the file and follow the directions in the terminal!
	1) If using CMD
		1) Copy the File Path
		2) Go into terminal and type 'cd {file path}'
		3) Then type 'python taskManagementMenu.py'
3) Follow the directions displayed in the console and enjoy!
